# What is an LLM

Token
- A short word, a syllable, a punctuation mark
- Unit of measure for input, processing, output
- Similar to 1st step taken by compilers
- Exists as vectors of floating point numbers

Large Language Model (LLM)
- Most likely next token statistical predictor
- Neural network based on transformer model
- Billions of floating point numbers (weights)

Pre-Training
- More "how to talk" than "what to say"
- Uses input data to determine weights for natural language
- Crunches trillions of tokens
- Data quality and selection is critical

Post Training
- Don't answer "what I asked", give me "what I wanted"
- Improves accuracy in specific domains
- Includes reinforce learning with human feedback (RLHF)
